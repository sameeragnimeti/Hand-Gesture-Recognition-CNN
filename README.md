# Hand Gesture Recognition using CNN

## Project Overview

This project implements a **Hand Gesture Recognition system** using **Convolutional Neural Networks (CNN)**.
The model is trained on the **LeapGestRecog dataset** to recognize and classify different hand gestures from images.

Hand gesture recognition is an important application of **Computer Vision** and **Human-Computer Interaction**, enabling systems to understand user gestures and respond accordingly.

## Dataset

The project uses the **LeapGestRecog dataset**, which contains thousands of labeled images representing different hand gestures.
Each gesture is stored in separate folders representing different classes.

## Technologies Used

* Python
* TensorFlow / Keras
* OpenCV
* Matplotlib
* Jupyter Notebook

## Model Architecture

The model is built using a **Convolutional Neural Network (CNN)** with:

* Convolution layers for feature extraction
* MaxPooling layers for dimensionality reduction
* Fully connected dense layers for classification
* Dropout for preventing overfitting

## Project Workflow

1. Load and preprocess the dataset
2. Split dataset into training and validation sets
3. Build CNN model architecture
4. Train the model on gesture images
5. Evaluate model accuracy
6. Visualize training results using graphs

## Results

The trained CNN model successfully classifies multiple hand gestures with good accuracy and demonstrates the effectiveness of deep learning in gesture recognition tasks.
